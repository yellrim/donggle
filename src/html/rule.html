동글서버 채팅 수칙

  • 전체 채팅을 통한 반말 금지

    + 실수일 경우도 있을 수 있으니 지속적인 반말 시 경고

  • 전체 채팅에 지속적인 도배 금지

  • 타 유저에게 혐오감을 주는 발언 및 성 드립은 경고없이 벤처리 당함을 이해 부탁드립니다.

  • 타 서버 유도 금지

  • 서버 분위기 선동 금지 = 관리자 판단하에 경고 지급 될 수 있습니다.

  • 비매너 채팅 및 욕설금지

동글서버 기타 수칙

  • 인게임 내의 밸런스를 파괴할 수 있는 핵(엑스레이), 모드(자동낚시), 모든 종류의 감마 사용 금지

    + 감마를 금지하는 이유는 게임 내에 야간 투시 아이템이 있기때문입니다. 적발 시 정지

  • 서버 내 알려지지 않은 버그를 제보 하지 않고 지속적으로 재화를 수급 하는 경우

  • 법전의 허점을 이용해 타 유저에게 악의적인 피해를 줄 시 관리자 판단하에 경고 지급(법전에 문제가 있는 부분이 있다면 제보 부탁드립니다.)

  • 타 유저에게 의도적으로 사망을 유발한 후 아이템을 얻을 시 관리자 판단하에 경고 또는 벤

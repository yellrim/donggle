동글서버 접속 방법

  • 자바에디션 1.20.1에서 원활한 한글 채팅을 이용하기 위해 아래 다운로드 버튼을 눌러 다운로드 해줍니다.(선택)

  • 한글채팅 다운로드
    패브릭 다운로드

  • 다운로드 후 Window + R키를 눌러 %appdata%를 입력 후 확인 버튼을 눌러줍니다.

  • .minecraft > mods 해당 경로에 들어오셔서 다운받은 CocoaInput(한글채팅)만 넣어줍니다.

  • 패브릭 파일은 다운받은 곳에서 더블클릭하여 버전을 1.20.1로 두고 install만 하면 됩니다.

  • 자바에디션 1.20.1 또는 위에 설치한 패브릭1.20.1로 접속합니다.

  • 멀티플레이로 접속합니다.

  • 우측 하단의 서버 추가를 클릭합니다

  • 서버 주소 dongle1.64bit.kr를 입력해줍니다.

  • 서버 리소스팩은 사용으로 변경해준 후 완료를 클릭하고 서버에 입장합니다.

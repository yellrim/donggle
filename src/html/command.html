동글서버 유저 명령어

  • /메뉴, Shift+F :

    메뉴창이 열립니다.

  • /spawn, /스폰 :

    서버의 스폰으로 이동합니다.

  • /장비숨김 :

    착용중인 장비가 숨겨집니다.

  • /chestsort :

    간편한 창고정리 설정

  • /수표 :

    본인의 소지금내에서 수표로 발행

  • /sit, 반블럭 우클릭 :

    원하는블럭 또는 반블럭 위에 앉을 수 있습니다.

  • /spin :

    행동, 모션

  • /ley :

    행동, 모션

  • /ps add [유저이름]:

    다른 유저에게 건차(건설차단) 권한을 줄 수 있습니다.

  • /귓 [유저이름]:

    해당 유저에게 귓속말 전송

  • /차단 추가 [유저이름]:

    귓속말을 받고싶지 않은 유저를 차단 합니다.

  • /차단 해제 [유저이름]:

    귓속말을 받고싶지 않은 유저를 차단 합니다.

  • /차단 목록:

    차단한 유저의 목록을 확인할 수 있습니다.

  • /위키:

    동글서버 위키 바로가기 링크 표시

  • /디스코드:

    동글서버 디스코드 바로가기 링크 표시
